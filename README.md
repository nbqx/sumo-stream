## Sumo Stream

[日本相撲協会公式サイト](http://www.sumo.or.jp/)取組表のReadableStream

## Usage

    $ git clone https://github.com/nbqx/sumo-stream.git
    $ cd sumo-stream
    $ npm install
    $ node cmd.js

or

    $ git clone https://github.com/nbqx/sumo-stream.git
    $ cd sumo-stream
    $ npm install -g .
    $ sumo-stream
